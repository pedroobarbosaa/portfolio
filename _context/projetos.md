# Projects

## 1. Netflix Analytics Pipeline
**Type:** Portfolio Project  
**Repo:** github.com/pedroobarbosaa/netflix-analytics-pipeline  
**Highlight:** Most technically complex project — end-to-end data engineering pipeline.

**What it does:**
Architected a full data pipeline from raw CSV ingestion to interactive dashboard:
CSV → Google Cloud Storage → BigQuery external tables → Star Schema (dim/fact tables) → 7 analytical views → Metabase dashboard via Docker.

**Technologies:** Python, GCP (BigQuery, Google Cloud Storage), SQL (SAFE_CAST, UNION ALL, SPLIT+UNNEST+CROSS JOIN, TIMESTAMP_SECONDS), Docker, Metabase, Star Schema

**Key achievement:** Production-grade analytical layer with automated GCS ingestion and advanced SQL patterns.

## 4. Tio André — Magician Website
**Type:** Freelance / Client Project
**Live:** tioandre.com.br
**Highlight:** Full client website built from scratch — design, copy, and deployment.

**What it does:**
Static website replacing a Wix site for a professional magician with 30+ years of career in Brasília. Built as a one-pager with dedicated commercial proposal subpages — each with a unique URL sent directly to clients via WhatsApp.

**Technologies:** HTML, CSS, JavaScript, Cloudflare Pages

**Design decisions:**
- Custom dark palette: Midnight Blue (#0A1628) + Electric Yellow (#F5C518) — conveys magic, nighttime show, premium feel
- Serif typography (Playfair-style) for headlines — elegance matching 30+ years of career
- Rich animations: parallax, animated counters, card fan effect on hero, scroll reveal
- Separate proposal pages (Festa Infantil, Dia das Crianças, Eventos) — each a standalone premium landing page

**Key achievement:** Delivered a complete brand presence — positioning, copy, visual identity, and deployment — replacing a generic Wix page with a custom site that reflects the client's premium market positioning.
